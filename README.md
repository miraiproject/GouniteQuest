## virtualenvwrapperのインストールと環境構築
```
$ pip install virtualenvwrapper
$ export WORKON_HOME=~/Envs
$ mkdir -p $WORKON_HOME
$ source /usr/local/bin/virtualenvwrapper.sh
$ mkvirtualenv quest
$ pip install -r requirements.txt
```

# GouniteQuest
GouniteQuestst3b3用のリポジトリです。

## ST3B3にて作成するアプリ
[ドキュメント](https://docs.google.com/document/d/1fZNyr7vXm150-iF55HSIN8XSBAOMuuF4onfwl4li9iA/edit)


## virtualenvwrapperのインストール
```
$ pip install virtualenvwrapper
$ export WORKON_HOME=~/Envs
$ mkdir -p $WORKON_HOME
$ source /usr/local/bin/virtualenvwrapper.sh
```

## 環境構築

```
$ git clone git@github.com:miraiproject/GouniteQuest.git
$ cd GouniteQuest
$ mkvirtualenv quest
$ pip install -r requirements.txt
```

# 各ページの説明
アプリ全体は以下のページから構成される。
+ （ヘッダー）
+ ユーザー登録ページ 
+ ログインページ
+ 生徒ページ
+ 先生ページ
+ 生徒の成績一覧ページ
+ 成績作成ページ
+ 成績変更ページ
+ レポート課題作成ページ
+ レポート課題詳細ページ兼レポート提出ページ
+ レポート課題編集ページ
+ プロフィール作成ページ
+ プロフィール編集ページ
+ 掲示板ページ

## ヘッダー
+ 「成績管理サイト」は生徒もしくは先生のトップページへのリンク。
+ 「~でログイン中」はログインしているユーザー名を表示。
+ 「ログアウト」はクリックでログアウトできる。
+ 「プロフィールを作成する」はプロフィール作成ページへのリンク。プロフィール作成済みのユーザーに対しては、「プロフィールを編集する」と表示し、プロフィール編集ページへのリンクとなる。

## ユーザー登録ページ
signup.html

ユーザー名、パスワード、パスワード（確認）を正しく記入することでユーザー登録ができる。

## ログインページ
login.html

ユーザー名とパスワードでログインできる。

## 生徒ページ
student_home.html

生徒用のトップページ。

主に以下の項目から構成される。

### 成績
自分の成績を確認できる。

項目は英語、数学、国語、GPA。

### レポート課題
レポート課題を確認できる。

項目はタイトル、課題作成者、締め切り日時、提出済みか。

未提出のものを上に表示し、締め切りが近いほど上に表示。

タイトルは課題詳細ページへのリンクになっており、レポートの提出もそこから行う。

### 提出したレポート
自分が提出したレポートの一覧を表示。

項目は、課題のタイトル、提出したファイル名、提出した日時。

新しく提出したものを上に表示。

ファイル名をクリックすると中身を確認できる。

### 掲示板
掲示板ページへのリンク

## 先生ページ
teacher_home.html

先生用のトップページ。

主に以下の項目から構成される。

### 生徒の成績
生徒の成績一覧ページへのリンク

### レポート課題
レポート課題を作成するボタンとレポート課題の一覧。

レポート課題を作成するボタンは、レポート課題作成ページへのリンク。

レポート課題の項目は、タイトル、課題作成者、締め切り、提出されたレポート数。

締め切りが近い順に表示。

タイトルはレポート課題詳細ページへのリンク。

### 提出されたレポート

生徒が提出したレポートの一覧を表示。

項目は、提出者、課題のタイトル、提出したファイル名、提出した日時。

新しく提出したものを上に表示。

ファイル名をクリックすると中身を確認できる。

## 生徒の成績一覧ページ
show_grade.html

平均点と生徒の成績一覧を表示。

### 平均点
英語、数学、国語それぞれの平均点を表示。

### 生徒の成績
生徒の成績一覧を表示。

項目は、生徒の名前、英語、数学、国語、GPA、成績をつけるボタン。

項目名をクリックすることで並べ替えをすることができる。

すでに成績がつけられている生徒に対しては「成績を変更する」と表示し、成績変更ページへのリンクとなっている。

まだ成績がつけられていない生徒に対しては、「成績をつける」と表示し、成績作成ページへのリンクとなっている。

## 成績作成ページ
new_grade.html

英語、数学、国語の得点を入力することで生徒の成績を保存することができる。

## 成績変更ページ
update_grade.html

英語、数学、国語の得点を入力することで生徒の成績を更新することができる。

## レポート課題作成ページ
show_report_problem.html

新しくレポート課題を作成するためのページ。

タイトル、内容、締め切りを入力することでレポート課題を作成できる。

## レポート課題詳細ページ兼レポート提出ページ
show_report.html

レポート課題詳細ページ。

生徒はレポートの提出を行うこともできる。

レポート課題を作成した先生は編集や削除ができる。



項目は、タイトル、課題内容、締め切り、レポート提出フォーム（生徒のみ）、編集・削除ボタン（課題作成者のみ）。

生徒はレポート提出フォームから、ファイルを選択し、レポートを提出することができる。

課題作成者は、編集ボタンでレポート課題編集ページに行くことができ、削除ボタンで課題を削除することができる。

## レポート課題編集ページ
update_report_problem.html

レポート課題を編集できる。

## プロフィール作成ページ
new_profile.html

生徒も先生も使用可能。

プロフィール作成ページ。

ログインしたユーザーが、まだ一度もプロフィールを作成していない場合にバナーのところに現れる。

記入できる項目は、プロフィールの画像と、紹介文の2つ。

保存後、作成した内容は、index.htmlの部分に表示される。

## プロフィール編集ページ
update_profile.html

生徒も先生も使用可能。

プロフィール編集ページ。

一度プロフィール作成した場合、バナーの部分にはプロフィール編集ページに飛ぶように変更されている。

記入できる項目は、プロフィールの画像と、紹介文の2つ。

保存後、編集した内容は、index.htmlの部分に表示される。
